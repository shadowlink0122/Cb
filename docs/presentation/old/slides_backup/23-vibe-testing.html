テスト駆動開発の重要性

仕様変更があっても壊れていることに気づきやすい

Cbのテスト戦略

  • 統合テスト : 3,000以上のCbコードのテストケース
  • ユニットテスト : C言語レベルでの関数テスト
  • 回帰テスト : 過去のバグの再発防止

テストの例

// tests/async_test.cb
async int test_sleep() {
    int start = time_ms();
    await sleep(100);
    int end = time_ms();
    return end - start >= 100;
}

void main() {
    assert(await test_sleep());
}

# テスト実行
$ make test
