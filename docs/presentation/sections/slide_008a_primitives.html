プリミティブ型とサイズ

型 サイズ（bit/byte） 説明
tiny 8 bit / 1 byte 符号付き8ビット整数
short 16 bit / 2 bytes 符号付き16ビット整数
int 32 bit / 4 bytes 符号付き32ビット整数
long 64 bit / 8 bytes 符号付き64ビット整数
float 32 bit / 4 bytes 単精度浮動小数点数
double 64 bit / 8 bytes 倍精度浮動小数点数
bool 8 bit / 1 byte 真偽値（true/false）
char 8 bit / 1 byte 文字型
void - 値なし（関数戻り値など）
string 可変 文字列型（動的サイズ）
T* 64 bit / 8 bytes ポインタ型（64bit環境）

型修飾子

  • unsigned: すべての整数型で使用可能（例: unsigned int）
  • const: 変数の再代入を禁止（例: const int x = 5;）
  • const T const*: ポインタの変更も禁止（二重const）
  • static: 静的変数として扱う
