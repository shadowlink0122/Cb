リテラル型 - TypeScriptから

特定の値のみを許可する型

// リテラル型の定義
typedef Status = "success" | "error" | "pending";

Status status;
status = "success";    // OK
status = "error";      // OK
// status = "unknown";  // コンパイルエラー！

// 数値リテラル型も可能
typedef Level = 1 | 2 | 3;
Level level = 2;  // OK
// level = 5;  // コンパイルエラー！

// 関数の戻り値にも使用可能
typedef Alignment = "left" | "right" | "center";
Alignment getAlignment() {
    return "center";
}

利点 : コンパイル時に不正な値を検出、タイポを防ぐ

使用例 : ステータス、設定値、列挙的な値の表現に最適
